# UNIV-SCHEDULER 🎓

Application de Gestion Intelligente des Salles et des Emplois du Temps à l'Université

## 📋 Description

UNIV-SCHEDULER est une application Java Swing qui permet de gérer les salles, les cours et les emplois du temps dans un environnement universitaire. Elle offre une interface intuitive pour différents profils d'utilisateurs : Administrateur, Gestionnaire, Enseignant et Étudiant.

## ✨ Fonctionnalités

- **Gestion des infrastructures** : Bâtiments, salles, équipements
- **Planification des cours** : Création, modification, suppression
- **Détection des conflits** : Alertes en cas de double réservation
- **Recherche de salles** : Filtres par capacité, équipement, disponibilité
- **Tableaux de bord** : Vues adaptées à chaque profil
- **Rapports** : Génération de statistiques d'occupation

## 🛠️ Technologies utilisées

- Java 25
- Swing (interface graphique)
- MySQL 
- GitHub
